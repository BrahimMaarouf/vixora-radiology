# Chest X-ray Taxonomy — Status + Type + Label Nature

Legend:
- Status: DONE / NOT DONE (based on your collected Kaggle sources)
- Type:
  - Disease (specific pathology)
  - Condition/Syndrome (state with multiple causes)
  - Radiological Finding (image pattern, not a diagnosis)
  - Injury/Trauma
  - Device/Artifact
  - Anatomical Variant/Deformity
- Label nature:
  - Single = can be treated as single-label (after filtering)
  - Multi = commonly co-occurs (multi-label in datasets)

# Ground Truth Summary

- Strong supervised (clean):  
  Pneumonia, Pneumothorax, Tuberculosis, Rib Fracture

- Weak / multi-label (NIH, VinBig):  
  Atelectasis, Effusion, Mass, Nodule, Fibrosis, Cardiomegaly, etc.
---

# Key Constraint

Dataset is **not disease classification**.  

It is:

> Chest X-ray pattern + condition detection (mixed ontology, multi-label reality)
